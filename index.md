# rime-wanxiang-notes

面向新手的「万象输入法 / rime_wanxiang」文档项目：用更清晰的目录、更短的路径，带你从 **0→可用→可定制→可进阶**。

> 本仓库是基于源项目内容整理的“二次文档工程”，不改动原方案实现；仅做更友好的讲解与索引。

## 你能得到什么

- **5 分钟上手**：复制文件 → 部署 → 选择方案 → 立刻开打
- **知道每个文件是干什么的**：`schema / dict / lua / opencc / custom patch` 一次讲清
- **基础 / 进阶分层**：先稳定可用，再逐步解锁高级玩法（辅助码、Tips、Lua、语法模型…）
- **可持续迭代**：为后续补充截图、平台差异、FAQ、最佳实践留好结构

## 推荐先点个 Star 的理由（写给“观望”的你）

1. 你会少走弯路：不会把配置写进 `custom/` 目录、不会被更新覆盖、不会“改了没生效”
2. 你会更快稳定：先把必需步骤跑通，再逐步尝试增强功能
3. 你会更敢改：每个改动都告诉你“改哪一个文件”“为什么是它”“如何回滚”

如果这套文档让你 **更快用上万象**、或者 **更敢折腾 rime**：欢迎 Star / 反馈 / PR。
