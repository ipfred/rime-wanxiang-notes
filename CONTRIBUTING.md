# 参与贡献

这个仓库只做“更好读、更好用”的文档整理：欢迎你用最小成本参与贡献（提 Issue 也算贡献）。

## 我们最需要的贡献

1. 新手卡点：你卡在哪一步？你以为应该怎么做？
2. 平台差异：Windows / macOS / Linux / Android / iOS 的“用户目录”“部署入口”“权限坑”
3. 亮点补充：某个功能很强，但新手不知道怎么触发/怎么配（请带最小示例）
4. 排错流程：把“遇到问题怎么定位”写成清单

## 写作约定（尽量统一）

- 先写“怎么做”，再写“为什么”
- 每页尽量解决一个问题：上手、切换方案、改候选、开 Tips、配置 patch…
- 提到配置文件时，明确它是“用户目录里的文件”，还是“源项目仓库里的样例文件”
