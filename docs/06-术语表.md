# 术语表（看到就不慌）

- **Rime**：输入法引擎（中州韵），不同平台有不同前端（小狼毫/鼠须管/fcitx/ibus…）。
- **前端**：承载 Rime 的“输入法应用”（UI、皮肤、部署入口、用户目录路径等由它决定）。
- **用户目录**：Rime 真正读取配置的目录，部署时会生成 `build/`、用户词库等。
- **schema**：输入方案（你在方案列表里勾选的那个），文件名一般是 `*.schema.yaml`。
- **dict**：词库入口文件（`*.dict.yaml`）以及分拆的词库数据（`dicts/`）。
- **custom patch**：`*.custom.yaml`，对 schema/default/前端配置的最终覆盖层，优先级最高。
- **部署（Deploy）**：让 Rime 重新加载并编译配置（把 YAML/Lua/词库等处理成可运行状态）。
