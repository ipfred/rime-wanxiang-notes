# 5 分钟上手（先跑起来）

目标：让你 **先能输入**，并能通过指令在全拼/双拼之间切换。

## 第 1 步：找到你的 Rime「用户目录」

最稳的方法（推荐）：在输入法托盘/菜单里找到类似入口：

- 小狼毫（Weasel）：右键图标 → “用户文件夹”
- 鼠须管（Squirrel）：菜单 → “用户设定”
- Fcitx5：输入法配置里找到 Rime 相关的“用户目录”
- ibus-rime：一般在设置里能找到“用户目录”

如果你一定要手动找（仅供参考，实际以你前端为准）：

- Windows（小狼毫）：`%APPDATA%\\Rime`
- macOS（鼠须管）：`~/Library/Rime`
- Linux（常见位置之一）：`~/.config/ibus/rime` 或 `~/.local/share/fcitx5/rime`

## 第 2 步：把万象方案文件放进用户目录

把“万象方案文件”（来源：源项目 Release / 方案包 / plum 安装得到的文件）复制到「用户目录」根目录（与 `default.yaml` 同级）：

- `*.schema.yaml`、`*.dict.yaml`、`*.yaml`（如 `wanxiang.schema.yaml`、`wanxiang.dict.yaml`、`wanxiang_symbols.yaml`…）
- `dicts/`、`lua/`、`opencc/`
- （可选）`weasel.yaml`（Windows 皮肤/行为）、`default.yaml`（全局默认设置）

## 第 3 步：部署（重新加载配置）

在你的输入法前端执行“重新部署/Deploy”。

部署完成后，方案列表里勾选 **「万象拼音」**（对应 `wanxiang.schema.yaml`）。

## 第 4 步：用指令切换输入方案（全拼 / 双拼）

在任意可输入文本的地方，直接输入以下指令（斜杠开头）：

```
/pinyin   → 全拼
/flypy    → 小鹤双拼
/zrm      → 自然码
/mspy     → 微软双拼
/sogou    → 搜狗双拼
```

输入后会提示“已切换…请重新部署”：照做再部署一次即可生效。

这些指令由 `lua/set_schema.lua` 实现：它会把示例 `*.custom.yaml` 复制到你的用户目录根目录，并把里面的转写段落改成对应方案。

## 第 5 步（建议）：确认你改配置的位置

后续所有自定义，优先改 **用户目录根目录**下的：

- `wanxiang.custom.yaml`（标准版）
- `wanxiang_pro.custom.yaml`（增强版/PRO）

不要在“方案包/源仓库”携带的 `custom/` 目录里直接改——那里是“样例与初始化来源”，不是最终生效位置。
